== Description ==

When Woocommerce updated to version 2.4 it broke a couple of our stores when the product had more than 20 variations. This plugin lets you increase the default number 10 to a higher number.  Hit save, and your variations will be back.

== Installation ==
Upload or install wc-variations-ajax.zip

Activate the plugin

Find the options page under Settings and input the desired amount
